🧠 Password Checker

Projeto consiste em um validador de força de senha em Python baseado em regras de segurança. A aplicação analisa a presença de números, letras maiúsculas e caracteres especiais utilizando métodos nativos da linguagem, e classifica a senha com base em um sistema de pontuação (fraca, média ou forte). O objetivo foi praticar manipulação de strings e implementação de regras de negócio.
